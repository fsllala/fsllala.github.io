NodeJs核心模块

Tips：核心模块：下载NodeJs自带的模块。

fs模块

文件操作都是基于fs模块。

  • readFileSync：同步读取(两个参数) 第二个参数可以是一个对象{encoding:’utf-8’};可以为字符串’utf-8’

  • writeFileSync：同步写入 文件写入的是字符串,所需需要JSON.stringify(); 写入操作 若文件名不存在,则自动创建。

    • r：只读；
    • w：写入(复写)；
    • a：写入(追加写)；

  • readdirSync：查看文件夹信息；这是查看一个文件夹中的所有文件夹或者文件，类似与doc命令中的dir；返回值是一个数组。

  • statSync：查看文件夹或者目录信息；主要用到了其 isFile()方法来判断是否为文件；返回值是一个对象。

  • mkdirSync：创建文件夹

  • rmdirSync：删除文件夹

  • unlinkSync：删除文件

  • renameSync：修改文件名

全局

url模块

  • 我们只使用其中的一个方法 目的是解析url；
  • 将一个完整的url解析成一个对象 使用url模块中的parse方法

querystring模块

  • 这个模块也是只有一个作用：用来处理get参数的；
  • 因为我们直接获取到的get参数是一个字符串 不能使用 name=Mary&age=19；
  • 我们可以借助querystring模块进行处理 使用模块中parse方法进行处理；
  • 将字符串格式的get参数 name=Mary&age=19 转化成对象格式 { name: ‘Mary’, age: ‘19’ }

path模块

path模块也是针对路径的模块 记住这些方法

  • basename：获取路径的最后一部分
  • dirname：获取除了页面部分
  • extname：获取后缀名
  • parse：获取一个对象 对象上述的所有
  • join：连接路径 相对路径
  • resolve：连接路径 绝对路径

http模块

http模块用来做web服务器，详情见下篇。
